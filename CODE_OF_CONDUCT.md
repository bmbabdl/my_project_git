# Code de Conduite

## Notre Engagement

Nous nous engageons à maintenir un environnement respectueux et accueillant.

## Comportements Attendus

- Faire preuve de respect envers tous les participants
- Être ouvert aux différentes opinions et expériences
- Accepter les critiques constructives

## Sanctions

Les comportements inappropriés peuvent entraîner des sanctions.
